# 星图 · 表情包工坊

一个纯浏览器运行的横幅表情包生成器，使用 React、TypeScript、TanStack Router、Vite 与 Canvas 构建。

## 功能

- 输入最长 12 个字符并实时预览
- 上传 TTF、OTF、WOFF、WOFF2 字体
- 上传 PNG、JPG、WebP、SVG Logo
- 调整文字厚底描边渐变、文字渐变与立体阴影
- 自动合并 Logo 与文字轮廓，并填满文字和 Logo 内部的封闭孔洞
- Logo 根据文字真实字形高度自动等高并居中，不需要手动调节
- 根据完整内容自动计算输出宽度，最长 464px，四周保留至少 1px 透明余量
- 使用 3 倍超采样和高质量缩放消除描边锯齿
- 提供 0–2px 可调 Alpha 羽化，为描边加入多级半透明过渡像素
- 厚底外增加 1px 白边与 1px 配色边，设置面板在同一页面内滚动
- 通过滑块调节字体大小与字重
- 通过 0–4px 滑块调节或关闭文字白色内描边
- 使用 120 组同色系与 120 组撞色渐变预设，支持搜索和每页 20 组分页
- 设置 240–464px 画布宽度
- 导出 PNG；高度固定为 96px，内容区为 94px，上下各保留 1px 透明余量
- 所有图片与字体均在浏览器本地处理
- 第二个 Tab 提供头像编辑器，任意尺寸图片可自动裁切为 1:1 正方形
- 支持缩放和调整头像裁切位置，以及独立开关上下弧形文字遮罩
- 可调整弧形遮罩高度、弧度、颜色、透明度和文字样式，并导出 800×800 PNG

## 开发

```bash
bun install
bun run dev
```

生产构建：

```bash
bun run build
```
